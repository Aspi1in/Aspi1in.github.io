概述

Java代码在计算机中经历的三个阶段：

  • 源代码阶段：Person.java –javac编译–> Person.class
  • 类加载阶段：Person.class –ClassLoader–> Class类型的对象来存储描述Person对象的信息
  • 运行时阶段：Person对象：new Person();

一个类文件一般会有三个域：Field、Constructor和Method，Class类主要就是负责描述这些信息的。
